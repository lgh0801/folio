    #React#Node.js#Mysql#Aws(ec2,rds)#REST API

    회원가입, 비밀번호 재설정, 로그인, 카카오(api) 로그인, 정보수정, 페이징, 검색
    에디터를 이용한 게시판 구현, 댓글과 대댓글 기능 구현
    토이프로젝트를 진행함에 있어서 함수형 컴포넌트와 클래스형 컴포넌트를 모두 사용해보며
    서로 다른 컴포넌트에 장단점을 체험했습니다. 또한 이 프로젝트는 컴포넌트가 많지 않아서 괜찮았지만
    부모 자식 간의 prop 전달이 component가 많아질 경우 점점 복잡해지므로
    redux사용을 중요하게 생각하는 이유를 느꼈습니다.
    이 프로젝트에서는 간단하게만 redux의 기능인 combineReducers,useDispatch,useSelector를 사용해 보았습니다.
    react를 처음 배우면서 작업을하여서 많이 부족하지만 저에게는 좋은 경험이 된 프로젝트였습니다.
